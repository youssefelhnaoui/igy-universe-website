Portfolio & Case Studies

SCCM Migration for Multi-Campus Organization

Directed a comprehensive migration of System Center Configuration Manager (SCCM) across multiple campuses, including server upgrades and cloud management gateway implementation. This project improved system manageability and security for over 10,000 users without disrupting daily operations.

Microsoft 365 Tenant Consolidation

Orchestrated the merger of 30 separate Microsoft 365 tenants into a single unified tenant. The consolidation streamlined IT administration and boosted collaboration across global teams, all while maintaining security and compliance throughout the process.

Active Directory Upgrade and Modernization

Led the upgrade of Active Directory from Windows Server 2008R2 to 2019 (alongside an Exchange 2010 to 2019 migration) for a large enterprise. This modernization enhanced system performance and security, providing a robust foundation for future cloud integration and growth.
